主題：Retrieval‑Augmented Generation（RAG）＋ 圖論（Knowledge Graph）＋ Agentic RAG

把「RAG 教學」做成能操作、能比較、能輸出報告的網站

這個專案是給學生做 RAG 上手的「教學坊」。每個功能都能做 A/B 比較（例如 Top‑K、重寫、圖譜、Agentic 等級），並把結果輸出成報告，方便複盤與教學。

GitHub Repo Demo 連結（待填） 看 A/B 與評估

三句話版本（給人資）

  • 把 RAG 常見調參（Top‑K/重寫/圖譜/Agentic）做成「可視覺化 + 可比較」的教學平台。
  • 有題庫、可編輯 UI、可匯入匯出，並能把評估輸出成 CSV/JSON（可再轉成 HTML 報告）。
  • 提供學生儀表板與練習建議區，讓學生能跟著做、做完能看差異。

快速驗證（5 分鐘內）

建議順序：先進管理員設定 → 上傳文件 → 去 RAG 教學坊做 A/B → 回到學生儀表板看練習建議。 
# 本機啟動
npm install
npm run dev

# build（預設使用 webpack）
npm run build
